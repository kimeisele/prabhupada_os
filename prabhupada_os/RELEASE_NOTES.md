## Overview
PrabhupadaOS is a zero-dependency, offline-first digital capsule for the Bhagavad-gita As It Is (1972 Edition). It combines a robust SQLite kernel with a 7-Dimensional Knowledge Hypercube to provide context-aware navigation and wisdom.

## System Stats
- **Verses**: 700/700 (100% Complete)
- **Integrity**: SHA-256 Verified Content Hashing
- **Knowledge Graph**: 7 Dimensions (Ontology, Process, Goal, Logic, Navigation, Interface)
- **Architecture**: Genesis Prompt (Cortex) + Zero-Dep Kernel + Vibe Connector

## Features
1.  **Offline Shell**: A robust CLI for browsing and searching the Gita without internet.
2.  **Intent Routing**: Automatically detects user intent (e.g., "depression") and routes to relevant chapters.
3.  **Semantic Context**: Pre-loaded theological structure to guide AI agents.
4.  **Immutability**: Every verse is cryptographically hashed to ensure the text remains "As It Is".

## Usage
To launch the offline shell:
```bash
python3 runtime/offline_shell.py
```
